# SwiftHelper
Swift中常用的Util和Extension

ios: 8.0+

###【安装】
```
pod 'SwiftHelper'
```

###【说明】

#####Utils
- InfiniteScrollView
两个UIImageView实现的无限滚动

- KeyboardManager
一行代码实现键盘监听

- AttributLabel
可以调整文本之间上下间距和左右间距

- ImageCropViewController
对图片裁剪

#####Extensions
- Array
- String
- NSDate
- NSTimer
- UIImage
- UIView
- UIViewController
- UIColor
